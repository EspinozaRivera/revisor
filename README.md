Requisitos
    Composer 2.5.*
    Laravel Framework 8.*
    php 7.*
    Apache 2.4.*

Instalacion
    composer install                        //1.- Instalar dependecias
    php artosam migrate:fresh --seed        //2.- Crear Tablas y agregar registros de preparacion (borra todo si ya existe)

Ejecutar servidor
    php artisan serve
